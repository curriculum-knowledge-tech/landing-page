Technical Curriculum

MERN

This course teaches the basics of web development, including HTML, CSS, JavaScript, and the MERN stack (MongoDB, Express, React, Node.js). It is suitable for beginners with some programming knowledge. You will learn to create responsive, dynamic web apps using React and manage global state with Redux. You will also learn to build server-side apps with Node.js and Express, and work with MongoDB for database management. By the end of the course, you will be able to develop a full-stack web app and deploy it to a live server.

Module 1: HTML, CSS, and JavaScript Basics

  • Introduction to HTML, CSS, and JavaScript
  • HTML tags and attributes
  • CSS selectors, properties, and values
  • JavaScript variables, data types, and functions
  • DOM manipulation with JavaScript
  • Introduction to responsive web design

Module 2: Introduction to React

  • Introduction to React and its features
  • Setting up a development environment with Node.js and npm
  • Creating and rendering components in React
  • JSX syntax and expressions
  • Component state and props
  • Handling user events and forms in React

Module 3: React Components and Lifecycle Methods

  • Component lifecycle methods in React
  • Conditional rendering and rendering arrays of data
  • Composing React components
  • Handling forms with controlled components
  • Higher-order components (HOCs)

Module 4: React Router and Redux

  • Introduction to React Router for client-side routing
  • Redux for global state management in React applications
  • Creating Redux stores, actions, and reducers
  • Connecting Redux to React components
  • Async actions with Redux Thunk middleware

Module 5: Introduction to Node.js

  • Introduction to Node.js and its features
  • Setting up a development environment with Node.js and npm
  • Node.js modules and package management
  • Node.js and the HTTP protocol
  • Creating a basic Node.js server

Module 6: Node.js with Express

  • Introduction to Express.js framework for Node.js
  • Handling requests and responses with Express
  • Express middleware for handling errors, authentication, validation and more
  • Using templates and rendering data with Express
  • Connecting to MongoDB database with Mongoose

Module 7: Advanced Database

  • Database schema design and modeling
  • Indexing and query optimization
  • Working with large datasets
  • Advanced Database aggregation

Module 8: Final Project

  • Planning and designing a full-stack web application
  • Implementing the application using the MERN stack
  • Using third-party APIs and libraries as needed
  • Testing and debugging the application
  • Deploying the application to a live server

Learning Outcomes

  • Develop a strong foundation in web development technologies such as HTML, CSS, and JavaScript
  • Build dynamic web applications using React, a popular JavaScript library for building user interfaces
  • Understand the component lifecycle methods in React and learn how to use them to manipulate the DOM, manage state, and handle user events and forms
  • Use React Router for client-side routing and Redux for global state management in React applications
  • Build server-side applications using Node.js and learn how to create basic Node.js servers
  • Work with Express.js, a popular Node.js framework, to handle requests and responses, use middleware, render data, and connect to MongoDB databases with Mongoose
  • Learn advanced database topics like schema design, indexing, query optimization, and working with large datasets
  • Plan, design, and implement a full-stack web application using the MERN stack and deploy it to a live server
  • Use third-party APIs and libraries to enhance the functionality of the application
  • Develop skills in testing and debugging web applications.

MEAN

Discover the power of MEAN stack, a leading full-stack development technology that empowers developers to build robust web applications using four powerful technologies: MongoDB, ExpressJS, AngularJS, and NodeJS. Our comprehensive course covers everything from the basics of JavaScript and NodeJS to advanced concepts like building RESTful APIs, using Angular for dynamic web pages, and integrating applications with MongoDB. Hands-on experience with each technology enables you to build complete applications and learn best practices for creating scalable, maintainable, and efficient MEAN stack applications.

Module 1: Introduction to MEAN stack

  • Setting up the development environment
  • Introduction to Node.js
  • Building a basic Node.js server and application

Module 2: MongoDB and RESTful APIs

  • Introduction to MongoDB
  • CRUD operations with MongoDB
  • Data modeling with Mongoose
  • Building a basic REST API with Node.js and MongoDB

Module 3: Express Framework

  • Introduction to Express framework
  • Routing and middleware in Express
  • Building a basic web application with Express and MongoDB

Module 4: Angular Framework

  • Introduction to Angular framework
  • Building Angular components, templates, and services
  • Data binding and event handling in Angular
  • Building a basic Angular application with Express and MongoDB backend

Module 5: Advanced Angular Concepts

  • Advanced Angular concepts: Reactive programming, Observables, RxJS
  • Building a real-time web application with Angular and Socket.io

Module 6: Authentication and Authorization

  • Authentication and Authorization with JSON Web Tokens (JWT)
  • User management and roles with Angular and Node.js
  • Building a secure MEAN stack application

Module 7: Deployment and Optimization

  • Deployment strategies for MEAN stack applications
  • Continuous integration and delivery
  • Performance tuning and optimization

Module 8: Advanced Topics and Final Project

  • Advanced topics: Serverless computing, Microservices architecture
  • Final project development and presentation

Learning Outcomes

  • Understand the basics of MEAN stack development and how to set up the development environment.
  • Gain proficiency in MongoDB, including CRUD operations, data modeling with Mongoose, and building RESTful APIs with Node.js and MongoDB.
  • Learn about the Express framework and its routing and middleware capabilities, and how to build web applications with Express and MongoDB. 
  • Master the Angular framework, including building components, templates, and services, data binding, and event handling, and how to build a basic Angular application with Express and MongoDB backend. 
  • Acquire knowledge of advanced Angular concepts such as Reactive programming, Observables, and RxJS, and how to build a real-time web application with Angular and Socket.io. 
  • Learn about authentication and authorization with JSON Web Tokens (JWT), user management, and roles with Angular and Node.js, and how to build a secure MEAN stack application. 
  • Understand deployment strategies for MEAN stack applications, continuous integration and delivery, and performance tuning and optimization. 
  • Explore advanced topics such as Serverless computing and Microservices architecture, and develop a final project to demonstrate the skills and knowledge acquired throughout the course.
